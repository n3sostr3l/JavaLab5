# JavaLab5 — Лабораторная работа №8 (GUI клиент)

## ТЗ (Техническое задание)

### Интерфейс
- Интерфейс должен быть реализован с помощью библиотеки **Swing**
- Графический интерфейс клиентской части должен поддерживать **4 языка / локали**:
  - Русский (`ru_RU`)
  - Белорусский (`be_BY`)
  - Албанский (`sq_AL`)
  - Английский (Индия) (`en_IN`)
- Должно обеспечиваться **корректное отображение чисел, даты и времени** в соответствии с локалью
- **Переключение языков** должно происходить **без перезапуска приложения**
- Локализованные ресурсы должны храниться в **файле свойств** (`.properties`)

### Замена консольного клиента на GUI
- Заменить консольный клиент на клиент с **графическим интерфейсом пользователя (GUI)**

### Функционал клиента

1. **Окно с авторизацией/регистрацией** ✅
2. **Отображение текущего пользователя** ✅
3. **Таблица, отображающая все объекты из коллекции**:
   - Каждое поле объекта — **отдельная колонка таблицы** ✅
   - Строки таблицы можно **фильтровать/сортировать** по значениям любой из колонок ✅
   - Сортировку и фильтрацию реализовать с помощью **Streams API** ✅
4. **Поддержка всех команд из предыдущих лабораторных работ** ✅
   - Все команды выполняются через **кнопки**
   - Каждая кнопка открывает **небольшое окно** (диалог) для ввода параметров
5. **Область, визуализирующая объекты коллекции**:
  - Объекты должны быть нарисованы с помощью **графических примитивов** (Graphics, Canvas) ✅
  - При визуализации использовать **данные о координатах и размерах объекта** ✅
  - Объекты от **разных пользователей** — **разными цветами** ✅
  - При **нажатии на объект** должна выводиться **информация об этом объекте** ✅
   - При добавлении/удалении/изменении объекта — **автоматическое появление/исчезновение/изменение** у всех клиентов ✅ (через auto-refresh)
   - При отрисовке объекта — **согласованная с преподавателем анимация** ❌ (отложено, пока делать не надо)
6. **Редактирование отдельных полей любого объекта (принадлежащего пользователю)**:
   - Переход к редактированию возможен **из таблицы** ✅
  - Переход к редактированию возможен **из области визуализации** ✅
7. **Возможность удаления выбранного объекта** ✅

## Структура проекта (GUI-часть)

```
src/main/java/com/akira/client/gui/
├── GuiClient.java            # Точка входа GUI-клиента
├── LocalizationManager.java   # Менеджер локализации
├── LoginFrame.java            # Окно авторизации/регистрации
├── MainFrame.java             # Главное окно приложения
├── LabWorkTableModel.java     # Табличная модель с сортировкой/фильтрацией
├── StreamsAPI.java            # Streams API для фильтрации/сортировки
├── VisualizationPanel.java    # Панель визуализации объектов
├── LabWorkDialog.java         # Диалог создания/редактирования LabWork
├── CommandDialogs.java        # Диалоги для всех команд
└── UIStyle.java               # Цветовая схема и стилизация

src/main/resources/i18n/
├── strings_en_IN.properties
├── strings_ru_RU.properties
├── strings_be_BY.properties
└── strings_sq_AL.properties
```

## Как собрать и запустить

```bash
# Сборка сервера и клиента
mvn package

# Запуск сервера
java -jar target/server.jar

# Запуск GUI-клиента
java -jar target/client.jar
```
